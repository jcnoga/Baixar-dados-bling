# Bling Intelligence

Sistema integrado com Bling ERP para análise de dados, CRM e gestão de estoque.

## 🚀 Funcionalidades

### 🔐 Autenticação
- Login com Firebase Authentication
- Perfis: Owner (admin) e Usuário comum
- Gerenciamento de usuários pelo Firebase Console

### 🔌 Integração Bling ERP
- **Importação Total**: Baixa todos os dados do Bling
- **Importação Incremental**: Baixa apenas registros novos ou alterados
- Mantém IDs originais do Bling
- Atualiza registros existentes

### 🏷️ Marcadores CRM (15 indicadores)
- **Clientes**: Inativos, VIP, Frequentes, em Queda
- **Produtos**: Campeões, Giro Lento, Sem Venda, Alta/Baixa Margem
- **Estoque**: Baixo, Alto, Parado
- **Oportunidades**: Promoção, Clientes Potenciais, Tendências

### 📊 Visualização de Dados
- Clientes, Produtos, Vendas, Compras, Fornecedores
- Exportação em JSON, CSV ou ZIP
- Dashboard com estatísticas

### 🧪 Dados Simulados
- Geração automática de dados para teste
- 10 registros por módulo
